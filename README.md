# TratorHub 🚜

TratorHub é uma plataforma web desenvolvida como **projeto pessoal**, com o objetivo de
praticar desenvolvimento **full stack** e criar uma solução real para anúncios de
compra, venda e aluguel de máquinas agrícolas.

A aplicação possui autenticação de usuários, persistência de dados e está publicada
em produção.

---

## 🔍 Funcionalidades

- Cadastro e autenticação de usuários (login)
- Criação e visualização de anúncios
- Integração com banco de dados
- Navegação entre páginas e fluxos de usuário
- Interface responsiva e organizada

---

## 🛠 Tecnologias Utilizadas

- **React**
- **TypeScript**
- **Vite**
- **Tailwind CSS**
- **Supabase** (back-end e banco de dados)
- **API REST**
- **Git & GitHub**
